Shopee desktop UI replica built with React + Vite for layout and interaction practice.

## Overview

This project recreates key desktop flows of Shopee with a focus on UI fidelity, responsive layout, and interactive components. It includes a mock authentication flow, product browsing, and a fully functional cart experience powered by Redux.

## Replicated Pages

- Home (banner, categories, limited-time deals, official store, popular items, daily discoveries)
- Login
- Signup
- Product detail
- Cart

## Key Features

- Fake login/logout with Redux state (no backend)
- Add to cart, quantity updates, and remove items
- Product detail view with pricing, rating, and shipping info
- Interactive carousels (Swiper) for banners and product strips
- Persistent header/footer and floating chat UI

## Tech Stack

- React + TypeScript
- Vite
- Redux Toolkit
- SCSS Modules
